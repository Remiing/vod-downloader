# VOD Downloader

SOOP의 VOD 및 클립을 가장 높은 화질로 다운로드하는 Python 도구입니다. 


## 주요 기능

* **파트 선택 다운로드**: 여러 개로 분할된 VOD 중 필요한 파트 번호만 입력하여 선택적으로 받을 수 있습니다.
* **자동 파일명 생성**: 스트리머 닉네임, 방송 날짜, 제목을 조합하여 깔끔한 파일명을 자동으로 생성합니다.


### 사전 요구 사항

**FFmpeg 실행 파일**: `ffmpeg.exe` 파일이 `main.py` (또는 빌드된 `.exe`)와 같은 폴더에 있어야 합니다.


### 로그인/성인 인증 (Cookie) 사용 방법

로그인이 필요한 VOD나 성인 제한 콘텐츠를 다운로드하려면 쿠키 설정이 필요합니다.

1.  브라우저에서 **[SOOP](https://www.sooplive.com)** 에 접속하여 로그인을 완료합니다.
2.  `F12` (개발자 도구)를 누른 뒤 상단의 `Console` 탭을 클릭합니다.
3.  입력창에 다음 명령어를 입력하고 엔터를 누릅니다.
    ```javascript
    document.cookie
    ```
4.  출력된 문자열 결과 위에서 마우스 오른쪽 버튼을 클릭하고 `Copy string contents` 를 선택합니다.
5.  프로젝트 루트 폴더(`main.py`와 같은 위치)에 `cookie.txt` 파일을 생성하고 복사한 내용을 붙여넣습니다.

`cookie.txt`에는 본인의 로그인 세션 정보가 담겨 있습니다. 이 파일이 외부에 유출되지 않도록 주의하세요.
