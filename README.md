# VOD Downloader

SOOP의 VOD 및 클립을 가장 높은 화질로 다운로드하는 Python 도구입니다. 


## 주요 기능

* **파트 선택 다운로드**: 여러 개로 분할된 VOD 중 필요한 파트 번호만 입력하여 선택적으로 받을 수 있습니다.
* **실시간 진행률 표시**: `rich` 라이브러리를 활용하여 콘솔에서 직관적인 프로그레스 바를 제공합니다.
* **자동 파일명 생성**: 스트리머 닉네임, 방송 날짜, 제목을 조합하여 깔끔한 파일명을 자동으로 생성합니다.


### 사전 요구 사항

**FFmpeg 실행 파일**: `ffmpeg.exe` 파일이 `main.py` (또는 빌드된 `.exe`)와 **같은 폴더**에 있어야 합니다.


### 사용 방법

1. URL 입력: 다운로드할 SOOP VOD 또는 클립 주소를 입력합니다.
2. 메타데이터 확인: 제목, 스트리머, 파트 구성 등 정보를 확인합니다.
3. 파트 선택: 다운로드할 파트 번호를 입력합니다. (예: 1 3 입력 시 1번과 3번 파트만 다운로드, 0 입력 시 전체 다운로드)
